🎮 3D OpenGL Game (Silk.NET)

A 3D technical demo built with C# and OpenGL. This project implements a custom 3D rendering pipeline from scratch without using a pre-made game engine.

🚀 Features

3D Rendering: Textured objects and GLSL shader integration.

Dual Camera: Toggle between First-Person and Third-Person views.

Physics: Basic collision detection and movement mechanics.

🛠 Tech

Language: C#

Graphics: OpenGL (Silk.NET)

Models: .obj format

🕹 Controls

WASD: Movement

C: Switch Camera
